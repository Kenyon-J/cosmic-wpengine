# Theme gallery

Ready-made layout themes for cosmic-wallpaper. To use one, either drop the
`.toml` file onto the Layout Themes page in Settings, or copy it into
`~/.config/cosmic-wallpaper/shaders/` and apply it there.

The format is documented in [docs/THEMES.md](../docs/THEMES.md) — every key
is optional and the Settings editor adjusts all of them live.

Contributions welcome: one `.toml` per theme, named after the theme, with a
short description added to the table below.

| Theme | Description |
| --- | --- |
| [centre-stage](centre-stage.toml) | Big circular art dead centre, ring visualiser around it, lyrics beneath |
| [minimal](minimal.toml) | Small corner art, thin bars along the bottom, no bounce |
